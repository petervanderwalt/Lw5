# Wishlist

- **DrawingBotV3 patterns** – support for importing DrawingBotV3-style generative sketch patterns (hatching, stippling, contour lines) as fill operations for vector shapes
- **SVGNest** – automatic nesting of parts within machine bounds to minimise material waste; integration with an SVGNest-like solver (or implement basic rectangle/polygon nesting)
- **Raster with space-filling curves** – raster engraving modes using standard space-filling curves (Hilbert, Peano, Moore, Gosper) and serpentine scan patterns, configurable per operation
- **Vectric-style offset tool** – interactive offset of selected paths with preview; video reference: https://www.youtube.com/shorts/We0WGl2aDAk
- Improved boolean operations on paths (union, difference, intersection) using Clipper2
- Multi-layer / multi-pass operations per document
- Custom tool library (laser, spindle, drag knife)
- DXF/DWG import improvements
- SVG filter support (stroke-to-path, bounding box clip)
- Undo/redo stack
- Persistent project files (.lw5 format)
- Machine-specific post-processor config UI
